# API (Application Programming Interface)

# Web Servis Mimarileri (SOA, REST, RESTFULL)

#SOA (Service Oriented Architecture)
SOA, Dağıtık bir yapıda bulunan servilere ait geliştirilen ve işlemleri bu servis üzerinden belirlnen bir geliştirme metodolojisidr. Ama her bir servisin kendine ait eylemlerini çakışmaya sebebiyet vermeden farklı kaynaklardan taleplerde bulunmasını sağlamak.

##Rest, Restfull
Rest, HTTP (Hyper text transfer protocol) protokolü üzerinden veri alışverişinin yapılmasını sağlayan bir mimaridir. Restfull Rest mimarinin yaptığı bütün işlemleri gerçekleştirir aynı zamanda protokol üzerinden birden fazla veri türünü de destekler.

#Restful formatları
- TEXT, XML, JSON, MEDIA

- Fiiler
	- GET, POST, PUT (Update), PATCH (Update), DELETE

## API STATUS CODE
- 100: Bilgilendirme
- 200: Başarı 
- 300: Yönlendirme
- 400: Client side hata kodları
- 500: Server side hata kodları
