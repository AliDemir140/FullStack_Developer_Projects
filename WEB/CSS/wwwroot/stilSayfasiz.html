CSS (Cascading Style Sheets)

    CSS yapısı özünde 3 farklı yaklaşım ile kullanılır.
  • Inline Css: ait olunan etiket içerisinde style attribute'nu kullanarak gerçekleştirilen işlem.
  • Internal Css:
    • ait olunan sayfa içerisinde kullanılan stil işlemleri.
  • External Css:
    • dış kaynaktan sayfaya aktarılan stil işlemleri.
