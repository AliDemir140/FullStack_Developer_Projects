# Infrastructure (Altyapı)
Infrastructure katmanı, uygulamanın dış dünya ile olan tüm entegrasyonlarını barındırır.

Application katmanının tanımladığı arayüzleri (Interface) gerçek sistemlere doğrudan bağlayan (adapter) yapıdır.

Kısaca;
Application Ne yapılacağını iletir.
Infrastructure nasıl yapılması gerekliliğini belirler.

# Infrastructure Katmanın Amacı
- veritabanı erişimi (EF Core)
- Repository Implementasyonları
- DbContext ve Configurations
- Migration
- Eposta, Dosya sistemi, 3rd party yazılımlar.
