# Application (Uygulama Katmanı)

Application katmanı, uygulamanın iş süreçlerinden sorumludur. İşlemleri yöneten beyindir.

Domain modeli nasıl çalışır, Infrastructure nasıl uygulari UI nasıl gösterir. bunların hepsini application belirler.


# Application Katmanın Temel Araçları

- iş kuralları application üzerinden uygulanır
- DTO lar burada oluşturulur.
- Validation'lar bu katmanda gerçekleştirilir.
- Hata/Başarı durumları da bu katmanda tanımlanır.

Not: hiç bir şekilde EF Core, SQl, HTTP
